# Talkito AI - Demo Page

 **Your Personal AI Interview Coach**

## About

This is the official demo/landing page for Talkito AI, launching in May 2026. Talkito AI is an advanced AI-powered interview preparation platform that provides real-time behavioral analysis, brutally honest feedback, and human-like AI interviewers.

## Features Showcase

-  **Real-Time Eye Contact Tracking** - MediaPipe tracks 478 facial landmarks
-  **4 AI Interviewer Personas** - Friendly Vandan, Empathetic Sarah, Strict David, Precise Shravani
-  **Brutally Honest Feedback** - No sugarcoating, just real improvement
-  **Voice & Anxiety Analysis** - Librosa-powered audio analysis
-  **Company-Specific Mode** - Practice for Google, Amazon, TCS, and more
-  **Anti-Cheat Detection** - Real proctoring features

## Tech Stack

- **Frontend**: Pure HTML/CSS/JavaScript
- **Design**: Glassmorphism, Modern Gradients, Responsive Layout
- **Fonts**: Google Fonts (Inter)
- **Colors**: Teal/Cyan Modern Theme
